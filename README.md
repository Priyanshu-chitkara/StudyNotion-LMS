# 🚀 StudyNotion LMS

An advanced Learning Management System (LMS) built using MERN Stack with authentication, OTP verification, instructor dashboard, recycle bin, course management, student enrollment, Cloudinary uploads, and responsive UI.

---

# 🌐 Live Demo

## Frontend
https://study-notion-lms-omega.vercel.app

## Backend API
https://studynotion-lms-1-8lfd.onrender.com

---

# ✨ Features

- 🔐 JWT Authentication
- 📧 OTP Email Verification
- 👨‍🏫 Instructor Dashboard
- 📚 Course Creation & Management
- 🎥 Video Upload Support
- 🖼️ Image Upload Support
- ☁️ Cloudinary Integration
- ♻️ Recycle Bin System
- 👨‍🎓 Student Enrollment
- 🌙 Dark Mode UI
- 📱 Responsive Design

---

# 🛠️ Tech Stack

Frontend:
- React.js
- Tailwind CSS
- Axios
- React Router

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer
- Cloudinary

Deployment:
- Vercel
- Render
- MongoDB Atlas
